# Support

Use this page for setup help and common next steps before opening an issue.

## Before Opening An Issue

1. Read [README.md](README.md).
2. Check [docs/runbooks/RUNBOOK.md](docs/runbooks/RUNBOOK.md) for bring-up and clean-machine validation guidance.
3. Check [docs/product/PROGRESS.md](docs/product/PROGRESS.md) to see whether the behavior is already a known in-progress area.

## Common Setup Questions

### The app installs, but I cannot chat

Make sure:

- Ollama is installed and running
- at least one supported local model is installed
- the Setup screen shows all required items as ready

### Voice is not working

Voice is currently optional.

For now, the app can still be considered healthy if text chat works and setup is otherwise complete.

### I am testing on a clean machine

Please note:

- whether Ollama was already installed
- whether any model was already downloaded
- whether this is a fresh Windows machine or a previously used one

That context makes setup reports much more useful.

## Where To File Things

- Feature requests: use the feature request template
