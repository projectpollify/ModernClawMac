# Code of Conduct

## Our Standard

We want ModernClaw to be a project where people can ask basic questions, share ideas, report problems, and collaborate without being mocked or dismissed.

Examples of behavior that help create that environment:

- being respectful and patient
- assuming good intent while still giving honest feedback
- focusing criticism on ideas, code, and behavior rather than people
- helping newer contributors get oriented
- keeping disagreements constructive and specific

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- insulting or belittling language
- deliberate trolling or hostile bad-faith argument
- publishing private information without permission
- repeated disruptive behavior after being asked to stop

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- project discussions
- review conversations
- other official ModernClaw community spaces

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject comments, commits, issues, or other contributions that do not align with this standard.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through the repository owner on GitHub and include as much context as you can.
